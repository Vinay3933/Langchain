#### The `/describe` command allows you to upload an image and generate four
possible prompts based on that image. Use the `/describe` command to explore
new vocabulary and aesthetic movements.

`/describe` generates prompts that are inspirational and suggestive, it cannot
be used to recreate an uploaded image exactly.  
`/describe` returns the aspect ratio for uploaded images.
